# Context-aware_LiDAR-forest-structure
This repository contains the data supporting the findings described in the research article: "**Context-aware UAV LiDAR reveals forest structure and improves tree diameter estimates in subalpine forest**" (Ecological Solutions and Evidence), by Revenga et al. (2026).
